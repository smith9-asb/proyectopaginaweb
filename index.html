México es lo mejor en su comida, disfruto mucho comiendo cuando he ido a visitar a mi familia alí. También apoyo bastante a la selección de fútbol de México. Mis Trabajos

En esta sección hablaré sobre el curso que estoy realizando y trabajos que ya he hecho. El curso es 1º ASIR (administrador de sistemas informáticos web). Como ya dije en la sección anterior, vine de bachillerato de ciencias y aunque haya dado asignaturas como TIC o Informática, todo lo que he estado dando es nuevo para mí. Aunque no he empezado tan bien en el primer trimestre, me gusta este curso ya que damos cosas que me interesan aprender, y no como asignaturas como lengua y literatura o historia. Algunas cosas que damos en el curso es aprender a ser un técnico de redes, o sobrellevar bases de datos o aprender a usar html y css como este caso, el cual he hecho esta página web con estos códigos.

-En la asignatura de redes usamos mucho la aplicación de Packet Tracer, en la que podemos simular sistemas de redes y cables con switches y ordenadores. Es importante en nuestro curso aprender como funcionan los sitemas de redes los cuales hacen que tengamos wifi, o conexiones entre ordenador y switches, etc.

-Después, otra asignatura muy importante es sistemas, en la que aprendemos los secretos de Windows o Ubuntu. Usamos mucho la terminal de ubuntu o en el caso de Windows, powershell o cmd. Probamos cosas como montar unidades, o crear particiones. Pero si queremos probar cualquier cosa sin saber si el resultado será el esperado, usamos virtualbox en el que creamos simulaciones de máquinas en las que podemos hacerle lo que queramos sin destrozar nuestro máquina principal.

-Finalmente, otra asignatura de las más importantes es Lenguaje de Marcas. Esta página web la he creado para esta asignatura, en la que he tenido que aprender sobre html y css para crear una página sobre mí. En esta asginatura, no solo aprendemos sobre estos códigos, también aprendemos sobre XML (en la que usamos aplicaciones como XML Copy Editor o Heidi). Pero si hablamos sobre este trabajo, he usado Visual Visual Studio Code. En esta página creas un archivo de html y otro de css, y los metes en una misma carpeta y ya tienes tu página web. Todavía estoy aprendiendo más sobre esta aplicación para poder perfeccionar mi página web.
